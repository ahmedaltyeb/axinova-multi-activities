

Build a complete corporate website UI for Axinova Multi Activities Company using React.



---



## 1. Project Goal



Build a high-quality bilingual corporate website UI for Axinova Multi Activities Company.



Axinova is a multi-sector organization in GCC and MENA.



Sectors:



* Industrial

* Trading

* Technical Services

* Logistics

* Construction



Target users:



* Investors

* Government clients

* B2B companies

* Industrial partners



Goal:

Create a professional corporate website UI that shows scale, trust, and enterprise identity.



---



## 2. Tech Stack (Lovable React Output)



Use:



* React (functional components)

* Tailwind CSS

* Framer Motion for animations

* Component-based architecture



Do not use:



* Next.js specific features

* Backend logic

* Database

* API routes



Keep output frontend only.



---



## 3. Design Rules



* Mobile-first design

* Responsive for all screen sizes

* Corporate and premium style

* Clean spacing and grid layout

* RTL support for Arabic + LTR for English

* Use logical spacing only (no left/right fixed layout assumptions)

* Modern enterprise UI style
